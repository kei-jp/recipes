# 作り置きスパイス味付け卵（標準）

## 位置づけ

卵を固ゆでにして殻をむき、スパイス入りの調味液に漬ける作り置き用の副菜である。
味をなじませておくことで、カレーの副菜にも単体のおやつにもそのまま使える。

## 標準レシピ

### できあがり

- 味付け卵6個分
- 調理時間: 20分程度（漬け込み時間を除く）
- 漬け込み時間: 冷蔵で6〜12時間
- 保存: 調味液ごと冷蔵し、3日以内を目安に食べ切る

### 材料

#### ゆで卵

| 材料 | 分量 | 役割・扱い |
| --- | ---: | --- |
| 卵 | 6個 | 高タンパクの主材料 |
| 水 | 卵が十分に浸かる量 | 加熱用 |
| 氷水または冷水 | 必要量 | 加熱後の冷却用 |

#### スパイス調味液

| 材料 | 分量 | 役割 |
| --- | ---: | --- |
| 醤油 | 60ml | 味付けのベース。カレーにも合わせやすい塩味 |
| 水 | 60ml | 醤油を薄め、卵を漬ける液量を確保する |
| 砂糖 | 小さじ1 | 醤油の角を取り、単体でも食べやすくする |
| コリアンダーパウダー | 1.5g | 丸い香りの土台 |
| クミンパウダー | 1g | カレーに合う香ばしさ |
| ブラックペッパー | 0.5g | 香りの輪郭と辛味 |
| ジンジャーパウダー | 0.5g | 後味の香り |
| カイエンペッパー | 0.1〜0.3g | 辛味。おやつ用は控えめにする |

醤油を使わない場合は、塩小さじ1弱を水120mlへ溶かして代用できる。ただし、味の厚みと色は変わるため別の試作として扱う。

### ゆで卵の手順

1. 鍋に卵を並べ、卵が2〜3cm浸かる量の水を加える。
2. 中火にかけ、沸騰したら弱火にして10〜11分加熱する。黄身まで固まった状態を標準とする。
3. 加熱後すぐに湯を捨て、氷水または冷水へ移す。
4. 5分以上冷やしてから殻をむき、水分を拭き取る。

### 調味液と漬け込み

1. 小鍋に水60ml、砂糖、コリアンダー、クミン、ブラックペッパー、ジンジャー、カイエンペッパーを入れて弱火にかける。
2. 砂糖が溶け、スパイスの香りが出るまで1〜2分温める。沸騰させすぎない。
3. 火を止めて醤油を加え、調味液を完全に冷ます。
4. 清潔な保存容器または食品用袋へ卵を入れ、冷ました調味液を注ぐ。
5. 卵全体が調味液に触れるようにし、冷蔵庫で6〜12時間漬ける。途中で一度上下を返すと味が均一になる。

> 温かい調味液へ卵を入れると傷みやすくなるため、必ず冷ましてから漬ける。

## 食べ方

### カレーの副菜

- 卵1個を半分に切り、カレーに添える。
- 漬け込み後の調味液が濃く感じる場合は、表面を軽く拭いてから使う。
- カレーへ崩して混ぜる場合は、1個を2食分に分けてもよい。

### おやつ

- 卵1個をそのまま食べる。
- 辛味が強い場合は、次回からカイエンペッパーを減らす。
- 単体で食べる用途では、漬け込み時間を6時間程度に抑えると塩味が重くなりにくい。

## 保存・衛生

- ゆで卵と調味液は、粗熱を取ってから清潔な密閉容器で冷蔵する。
- 冷蔵保存は3日以内を目安にし、取り分けには清潔な器具を使う。
- 調味液から取り出した卵を常温に長時間置かない。
- 使い終わった調味液は再利用しない。
- 再加熱はせず、冷たいまま食べる。異臭、変色、ぬめりなどがある場合は食べない。

## 調整方法

- 味が濃い場合は、漬け込み時間を短くする。調味液を薄める場合は醤油と水の比率を保つ。
- 甘い香りを強める場合は、コリアンダーを0.5g増やす。
- カレー感を強める場合は、クミンを0.5g増やす。
- 辛味が強い場合は、カイエンペッパーを減らし、ブラックペッパーも必要に応じて減らす。
- 魚や豆のカレーに合わせる場合は、ジンジャーを少し増やす案を試す。

## 置き換えと購入提案

- ブラックペッパーはホワイトペッパーに一部置き換えられる。香りが穏やかになり、単体のおやつ用に向く可能性がある。
- 醤油以外の調味料を使う場合は、塩分濃度と漬け込み時間を別途評価する。
- リスト外のスパイスを試す場合は、必須ではなく提案として扱い、カレーとの相性・辛味・保存性への影響を確認する。

## 設計意図

### 味付け液に漬ける理由

食べる直前に粉末を振る方式よりも味がなじみやすく、卵だけで副菜やおやつとして成立しやすい。スパイスを調味液へ加えて温めることで、粉末の香りを液体へ移す。

### 固ゆでを標準にする理由

作り置き時の扱いやすさを優先し、黄身まで固まる状態を標準にする。半熟に変更する場合は、保存期間と取り扱い条件を別途評価する。

### カレー副菜とおやつの両立

醤油のうま味でカレーに合わせやすくし、砂糖と控えめな辛味で単体でも食べやすくする。漬け込み時間を変えることで、用途に合わせて塩味を調整できる。

## 初回評価項目

- 黄身まで固まった固さが食べやすいか
- 6〜12時間の漬け込みで味が濃くなりすぎないか
- カレーに添えたとき、醤油とスパイスの香りが強すぎないか
- 卵単体のおやつとして食べやすいか
- 調味液を作って漬ける工程が続けやすいか
- 冷蔵3日目の卵の食感と香りが許容範囲か
- 次回に変更する要素を1つに絞れるか

初回調理の所感と改善内容は、調理後に `history/` 配下へスナップショットとして記録する。
