# 万能ガラムマサラ PoC（標準）

## 位置づけ

カレーを食べる直前に少量振りかけ、香りを足すための試作配合である。
肉・豆・野菜のカレーに合わせやすいよう、辛さや色よりも甘い香りと温かい香りを優先する。

仕上がり量は約25gとし、まずは少量で香りの強さと使いやすさを評価する。
スパイスは [手持ちスパイス](../../ingredients/spices.md) を優先し、ホールスパイスの粉砕工程は必須にしない。

## PoC 配合

### 仕上がり

- 約25g
- 用途: 完成したカレーへの仕上げがけ
- 1食あたりの使用量: 0.5〜1g（目安として小さじ1/4〜1/2弱）
- 25gで約25〜50食分を想定

### 材料

| スパイス | 分量 | 役割 |
| --- | ---: | --- |
| コリアンダーパウダー | 8g | 甘く丸い香りの土台 |
| クミンパウダー | 6g | 香ばしさとカレーらしい骨格 |
| ブラックペッパー | 3g | 香りの輪郭と軽い辛味 |
| カルダモンパウダー | 3g | 甘く華やかな香り |
| シナモンパウダー | 2g | 甘い香りと温かみ |
| ジンジャーパウダー | 2g | 余韻と軽い辛味 |
| クローブパウダー | 1g | 奥行き。強く出やすいため少量 |
| **合計** | **25g** |  |

### 手順

1. 乾いた小皿またはボウルに、すべてのスパイスを重量で量る。
2. 小さな泡立て器またはスプーンで、色と香りが均一になるまでよく混ぜる。
3. 混ぜた直後に少量を手のひらで温め、香りを確認する。
4. 乾燥した密閉容器へ移し、作成日と配合名を記録する。

粉末スパイスのみで作るため、焙煎や粉砕は行わない。香りが弱い場合の次の試作で、クミンシードなどホールスパイスの乾煎り・粉砕を検討する。

## 使い方

- カレーを器に盛り、食べる直前に1食あたり0.5gから振りかける。
- 香りが足りない場合だけ、少量ずつ追加する。
- 鍋全体へ混ぜ込まず、同じカレーで使用量を比較できるようにする。
- 最初は肉・豆・野菜のカレーにそれぞれ同量を試し、香りの相性を比べる。

### 使用量の目安

| 用途 | 1食あたり | 使い方 |
| --- | ---: | --- |
| 香りを軽く足す | 0.5g程度 | ひと振りから開始 |
| 標準的に使う | 0.75g程度 | 食べる直前に全体へ振る |
| 香りを強くする | 1gまで | 味を確認しながら追加 |

## 配合の調整

- 甘い香りを強める場合は、カルダモンを0.5〜1g増やし、同量のコリアンダーを減らす。
- 香ばしさを強める場合は、クミンを1g増やし、同量のコリアンダーを減らす。
- 辛味や刺激が強い場合は、ブラックペッパーまたはジンジャーを減らす。
- 薬品のような香りが出た場合は、クローブを半分にする。
- 香りが弱い場合は、使用量を増やす前に、スパイスの鮮度と保管状態を確認する。

ターメリックとカイエンペッパーは、色や辛味が主目的になるため基本配合には含めない。必要なカレーだけ、別途個別に調整する。

## 保存

- 完全に乾いた密閉容器へ入れる。
- 湿気、直射日光、高温を避け、冷暗所で保管する。
- 濡れたスプーンを容器へ入れない。
- 香りを重視する PoC のため、作成後1か月程度を香りの評価期間の目安とする。
- 香りが明らかに弱くなった、湿気て固まった、異臭がある場合は使用せず、次の配合を作る。

## 設計意図

### 仕上げ用途に絞る理由

煮込み中ではなく食べる直前に使うことで、揮発しやすい香りを残しやすくする。鍋全体へ入れず、使用量を比較できるようにする。

### 香りの軸

コリアンダーとクミンでカレーに合わせやすい土台を作り、カルダモン・シナモン・クローブで甘く温かい香りを加える。ブラックペッパーとジンジャーは少量に抑え、料理の種類を選びにくくする。

### 少量試作にする理由

配合済みスパイスは、実際に料理へ振りかけるまで相性を判断しにくい。25gに抑え、複数のカレーで試してから標準配合を固定する。

## 初回評価項目

- 肉・豆・野菜のカレーで香りの相性がよいか
- 食べる直前に振りかけたとき、香りが十分に立つか
- カルダモン、シナモン、クローブが強すぎないか
- 魚介や辛味のあるカレーでも料理を邪魔しないか
- 0.5〜1gの範囲で使いやすいか
- 1か月保管後も香りを維持できるか
- 次回に変更する要素を1つに絞れるか

初回配合の所感と改善内容は、調理・試食後に `history/` 配下へスナップショットとして記録する。
