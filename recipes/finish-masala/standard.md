# 鯖スープ・豆スープ用仕上げマサラ PoC（標準）

## 位置づけ

鯖スープや豆スープを器に盛った後、食べる直前に少量加えて香りを引き立てるための試作配合である。
魚臭さを強めず、豆の風味も残せるよう、万能ガラムマサラより甘い香りとクローブを抑え、コリアンダー・生姜・胡椒を中心にする。

仕上がり量は約15gとし、まずは鯖スープと豆スープで相性を比較する。
スパイスは [手持ちスパイス](../../ingredients/spices.md) を優先し、追加購入を前提にしない。

## PoC 配合

### 仕上がり

- 約15g
- 用途: 鯖スープ・豆スープへの仕上げがけ
- 1食あたりの使用量: 0.25〜0.5g（ひとつまみ〜小さじ1/4弱）
- 15gで約30〜60食分を想定

### 材料

| スパイス | 分量 | 役割 |
| --- | ---: | --- |
| コリアンダーパウダー | 6g | 丸く軽い香りの土台。魚・豆の両方に合わせる |
| ブラックペッパー | 3g | 香りの輪郭と軽い辛味 |
| ジンジャーパウダー | 2g | 鯖の風味を整え、後味を軽くする |
| ホワイトペッパー | 2g | スープになじむ穏やかな刺激 |
| クミンパウダー | 2g | 豆とスパイスのつながり。入れすぎない |
| **合計** | **15g** |  |

### 手順

1. 乾いた小皿またはボウルに、各スパイスを重量で量る。
2. 小さな泡立て器またはスプーンで、色と香りが均一になるまで混ぜる。
3. 少量を手のひらで温め、魚向けとして刺激や香りが強すぎないか確認する。
4. 乾燥した密閉容器へ移し、作成日と配合名を記録する。

粉末スパイスのみで作るため、焙煎や粉砕は行わない。香りが弱い場合は、次回試作でクミンシードの乾煎り・粉砕や、レモン系の香りを加える案を検討する。

## 使い方

- スープを温めて器に盛った後、食べる直前に0.25gから振りかける。
- 最初は器の片側だけに加え、加える前後の香りと味を比較する。
- 鍋全体へ混ぜ込まず、鯖スープと豆スープで使用量を比較できるようにする。
- 香りが足りない場合だけ、0.1gずつ追加する。

### 使用量の目安

| 用途 | 1食あたり | 使い方 |
| --- | ---: | --- |
| 軽く香りを足す | 0.25g程度 | ひとつまみから開始 |
| 標準的に使う | 0.4g程度 | 器全体へ薄く振る |
| 香りを強くする | 0.5gまで | 魚臭さや辛味を確認しながら追加 |

## スープ別の調整

### 鯖スープ

- まず0.25gから使い、魚臭さが強くならないかを確認する。
- 生姜とブラックペッパーが強く感じられる場合は、次回配合でそれぞれを減らす。
- 香りが魚に負ける場合でも、クミンを大きく増やさず、コリアンダーを1g増やす案を先に試す。

### 豆スープ

- 0.4g程度から使い、豆の甘みとコリアンダーの香りがつながるかを確認する。
- 物足りない場合はクミンを1g増やす案を試す。
- 粘度の高い豆スープでは粉が残りやすいため、少量のスープで溶いてから加えてもよい。

## 配合の調整

- 魚臭さを抑えたい場合は、生姜を1g増やす前に、スープ本体の温度と保存状態を確認する。
- 刺激が強い場合は、ブラックペッパーとホワイトペッパーを合計1g減らし、コリアンダーを同量増やす。
- クミンの香りが強い場合は、1g減らしてコリアンダーへ置き換える。
- 甘い香りを追加するためのカルダモン、シナモン、クローブは、魚との相性を見てから少量の任意案として検討する。

## 保存

- 完全に乾いた密閉容器へ入れる。
- 湿気、直射日光、高温を避け、冷暗所で保管する。
- 濡れたスプーンを容器へ入れない。
- 香りを重視する PoC のため、作成後1か月程度を評価期間の目安とする。
- 湿気て固まった、異臭がある、香りが明らかに弱くなった場合は使用せず、次の配合を作る。

## 設計意図

### 魚臭さを邪魔しないための配合

カルダモン・シナモン・クローブのような甘く強い香りは基本配合から外し、コリアンダーと生姜を中心にした。ブラックペッパーとホワイトペッパーは刺激が重なりすぎないよう、合計量を抑えている。

### 食べる直前に使う理由

煮込み中に加えると香りが弱くなり、魚や豆の味に埋もれやすい。器へ盛った後に使い、少量ずつ香りの変化を確認する。

### 万能ガラムマサラとの差

万能ガラムマサラは肉・豆・野菜カレー全般の香り付けを想定する。一方、この配合はスープの水分と魚の風味を前提に、甘い香りとクローブを抑え、少量で後味を整えることを優先する。

## 初回評価項目

- 鯖スープで魚臭さを強めないか
- 鯖スープの生姜・和風だし・トマトと相性がよいか
- 豆スープで豆の甘みを邪魔しないか
- 0.25〜0.5gの範囲で香りの変化を感じられるか
- ブラックペッパーとホワイトペッパーの刺激が強すぎないか
- 1か月保管後も香りを維持できるか
- 次回に変更する要素を1つに絞れるか

初回試作の所感と改善内容は、調理・試食後に `history/` 配下へスナップショットとして記録する。
