# 鶏むね肉の炊飯器保温調理（標準）

## 位置づけ

鶏むね肉を調味液に漬け込み、炊飯器の保温モードで加熱する標準レシピである。
漬け込み時間など、今後検証できる条件は調理履歴に記録し、改善時はこの `standard.md` を更新する。

## 分量

### 材料

| 材料 | 分量 | 用途・扱い |
| --- | ---: | --- |
| 鶏むね肉 | 約300g（1枚） | 全体を肉たたきやフォーク等で刺す |
| 水 | 100cc | 調味液に使用 |
| 液体塩こうじ | 40cc | 調味液に使用 |
| 白だし | 20cc | 調味液に使用 |
| 砂糖 | 小さじ1 | 調味液に使用 |
| 乾燥こんぶ | 1枚 | 鶏むね肉・調味液と一緒に袋へ入れる |
| 耐熱ビニール袋 | 1枚 | 鶏むね肉と調味液を入れる |

### 調理条件

- 漬け込み時間: 約24時間
- 加熱方法: 炊飯器の保温モード
- 加熱時間: 90分

## 手順

1. 水、液体塩こうじ、白だし、砂糖を混ぜて調味液を作る。
2. 鶏むね肉全体を、肉たたきやフォーク等で十分に刺し、味が入りやすいようにする。
3. 耐熱ビニール袋へ鶏むね肉、調味液、乾燥こんぶを入れる。
4. 袋を冷蔵庫に入れ、約24時間漬け込む。
5. 炊飯器へ水を張り、調味液と鶏むね肉を入れた袋を沈める。
6. 炊飯器を保温モードにし、90分加熱する。
7. 加熱後は速やかに袋ごと取り出し、流水などで粗熱を取る。
8. 鶏むね肉をスライスし、保存容器へ移して保存する。

## ポイント

- 漬け込み時間は現時点では約24時間とする。数時間でも問題ない可能性はあるが、短時間での仕上がりは未検証である。
- 鶏むね肉全体を十分に刺して、調味液が入りやすい状態にする。
- 耐熱性のある袋を使用し、袋が炊飯器の熱源に直接触れないようにする。
- 加熱後は速やかに粗熱を取り、保存容器へ移す。
- 鶏肉の中心部まで十分に加熱されていることを確認する。
