# 複数スパイスのチャイ

## 調理日

2026-07-20

## 位置づけ

カルダモン、ジンジャー、シナモン、クローブを組み合わせ、複数スパイスによるチャイらしい香りと各スパイスの役割を確認した試作。

## 材料

| 材料 | 分量 | 備考 |
| --- | ---: | --- |
| 水 | 約500mL |  |
| 牛乳 | 約400mL |  |
| アッサム茶葉 | 小さじ4 |  |
| カルダモン | 小さじ1 |  |
| ジンジャー | 小さじ1/2 |  |
| シナモン | 小さじ1/4 |  |
| クローブ | 小さじ1/8 |  |

## 調理方法

1. 水にカルダモン、ジンジャー、シナモン、クローブを入れ、2〜3分煮る。
2. 茶葉を加え、2〜3分煮出す。
3. 牛乳を加える。
4. 沸騰させない程度で5分ほど温める。
5. 茶こしでこす。

## 所感

- 複数スパイスを組み合わせることで、一気に「チャイらしい」複雑な香りになった。
- シナモンは単体では支配的だが、煮出すと全体へ溶け込む。
- クローブは前へ出るよりも余韻を作る印象だった。
- ジンジャーは刺激というより温かみを加える。
- カルダモンは全体を爽やかにまとめる役になった。

## 学び

単体では強く感じるスパイスも、組み合わせて煮出すことで役割が変わる。シナモンやクローブの量を増やす場合は、カルダモンの爽やかさと全体の余韻を損なわない範囲で調整する。
