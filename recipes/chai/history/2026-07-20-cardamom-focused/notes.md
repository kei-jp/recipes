# カルダモンを知るチャイ

## 調理日

2026-07-20

## 位置づけ

カルダモン単体の香りと、温度による感じ方の違いを確認するための1杯分の試作。

## 材料

| 材料 | 分量 |
| --- | ---: |
| アッサム茶葉 | 小さじ2 |
| 水 | 150cc |
| 牛乳 | 150cc |
| カルダモンパウダー | 小さじ1/2 |
| 砂糖 | 小さじ1 |

## 調理方法

1. 水にカルダモンパウダーを入れ、中火にかける。
2. 沸騰してから2分程度、カルダモンを抽出する。
3. 茶葉を加え、2分程度抽出する。
4. 牛乳を加えて温める。
5. 沸騰直前で火を止める。
6. 茶こしでこし、砂糖を加える。

## 所感

- カルダモンは爽やかで心地よい。
- 想像より主張は強くない。
- 温かい状態では香りを掴みにくい。
- 氷で冷やすとカルダモンの香りが分かりやすくなった。
- シナモンを後がけすると、シナモンに完全に支配されることを確認した。

## 学び

- カルダモンは小さじ1/2程度で十分と感じた。
- スパイスは温度によって印象がかなり変わる。
- スパイスを組み合わせる前に、まず単体の香りを知ることが重要。
