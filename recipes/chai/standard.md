# 毎日飲める甘さ控えめチャイ（標準）

## 位置づけ

手持ちのアッサム茶葉と粉末スパイスで、1杯分を短時間に作る日常用チャイである。
牛乳を主役にし、市販品より甘さを控えめにして、カルダモンを中心としたスパイスの香りを楽しむ。

チャイ専用マサラは別レシピとして検討するため、この標準レシピではスパイスを毎回個別に加える。

## 標準レシピ

### できあがり

- 1杯分（約250ml）
- 調理時間: 8〜10分程度
- 洗い物: 小鍋、茶こしまたは細かいザル、カップ

### 材料

| 材料 | 分量 | 役割・扱い |
| --- | ---: | --- |
| アッサム茶葉 | 5g | 濃いめに煮出す。ティーバッグなら1個でもよい |
| 水 | 100ml | 茶葉とスパイスを煮出す |
| 牛乳 | 180ml | 主体となる液体。成分無調整を基本とする |
| 砂糖 | 小さじ1（約4g） | 甘さ控えめの基準。好みで増減する |
| カルダモンパウダー | 小さじ1/4 | 主役となる甘い香り |
| シナモンパウダー | 小さじ1/8 | 甘い香りの補助 |
| クローブパウダー | ひとつまみ | 奥行き。入れすぎない |
| ジンジャーパウダー | ひとつまみ | 余韻の辛味と香り |

### 手順

1. 小鍋に水、カルダモン、シナモン、クローブ、ジンジャーを入れて中火にかける。
2. 沸騰したら弱火にし、1分ほど煮てスパイスの香りを水へ移す。
3. アッサム茶葉を加え、弱火で2分煮出す。渋みが出やすいため、長く煮すぎない。
4. 牛乳を加えて中火に戻す。沸騰させすぎないよう、鍋肌がふつふつする程度まで温める。
5. 弱火で2〜3分温め、牛乳に茶葉とスパイスの香りを移す。
6. 火を止め、砂糖を加えて溶かす。
7. 茶こしでこしながらカップへ注ぐ。

> 牛乳を加えた後に強く沸騰させると、膜ができたり風味が重くなったりしやすい。沸騰直前を保つ。

## 甘さ・スパイスの調整

### 甘さ

- 標準は砂糖小さじ1とし、甘さ控えめにする。
- 甘さをさらに抑える場合は小さじ1/2、無糖にする場合は砂糖なしにする。
- 物足りなさを甘さで補わず、カルダモンまたはシナモンを少量だけ増やす。

### スパイス

- スパイス感を強める場合は、カルダモンを小さじ1/3まで増やす。
- 香りを弱める場合は、クローブとジンジャーを先に減らす。
- クローブは少量でも強く出るため、毎日飲む場合はひとつまみを超えない。
- 辛味が必要な場合は、ブラックペッパーを少量追加する。標準では必須にしない。

## 毎日続けるための運用

- 茶葉と粉末スパイスは、1杯分を先に小皿へ量っておくと調理が早い。
- スパイスを水から煮出すことで、牛乳を長時間沸騰させずに香りを出せる。
- 粉末スパイスは完全には溶けないため、茶こしを使う。細かい粉が残ることを許容するなら、こさずに飲んでもよい。
- 牛乳を加えた後は目を離さず、吹きこぼれを防ぐ。
- 洗い物を減らしたい場合は、茶葉をティーバッグに入れて煮出す。

## 置き換えと購入提案

- アッサム茶葉は、濃いめに出せる他の紅茶でも代用できる。香りと渋みは変わるため、茶葉量で調整する。
- 牛乳は豆乳などに置き換えられるが、コクと加熱時の分離しやすさが変わるため別途評価する。
- リスト外のスパイスを使うと香りが改善する場合は、必須ではなく提案として扱う。

## 失敗回避

- 茶葉を長く煮すぎない。渋みが強い場合は煮出し時間を短くする。
- 牛乳を強火で沸騰させない。鍋底が焦げそうなら一度火を弱める。
- クローブを増やしすぎない。薬品のような香りが出た場合は次回量を半分にする。
- 粉末スパイスが沈む場合は、注ぐ前に茶こしでこすか、飲む前に軽く混ぜる。

## 設計意図

### アッサム茶葉を使う理由

アッサムは牛乳に負けにくい濃さを出しやすく、少量でもチャイらしい茶の土台を作りやすい。手持ちの茶葉を使うことで、毎回の購入を増やさず継続できる。

### 甘さを控える理由

砂糖を小さじ1に固定した基準を作り、甘さではなくカルダモンとシナモンの香りで満足感を補う。毎日飲む前提で、飲み終わりが重くならないことを優先する。

### 粉末スパイスを使う理由

手持ちの粉末スパイスをそのまま使い、ホールスパイスの準備や粉砕工程を増やさない。香りの立ち方はホールスパイスと異なるため、必要になった場合に次の改善候補として評価する。

## 初回評価項目

- アッサム茶葉5gで牛乳に負けない濃さになるか
- 甘さ控えめでも満足感があるか
- カルダモンの香りが主役になっているか
- クローブやジンジャーが強すぎないか
- 8〜10分の工程が毎日続けられるか
- 茶こしを含む洗い物が負担にならないか
- 次回に変更する要素を1つに絞れるか

初回調理の所感と改善内容は、調理後に `history/` 配下へスナップショットとして記録する。
