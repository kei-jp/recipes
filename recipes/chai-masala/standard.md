# カルダモン主体のチャイ専用マサラ PoC（標準）

## 位置づけ

毎日作るチャイへ少量加えるための専用マサラである。
カルダモンを主役にし、シナモンとジンジャーで甘い香りを補う。クローブとブラックペッパーは控えめにして、毎日飲んでも重く感じにくい配合を目指す。

チャイ本体は [毎日飲める甘さ控えめチャイ](../chai/standard.md) を基本とし、このマサラを使う場合は本体レシピの個別スパイスを置き換える。

## PoC 配合

### 仕上がり

- 約15g
- 1杯あたりの使用量: 0.4〜0.6g
- 約25〜37杯分
- 用途: 牛乳ベースのチャイへの加熱時投入

### 材料

| スパイス | 分量 | 役割 |
| --- | ---: | --- |
| カルダモンパウダー | 8g | 主役。甘く華やかな香り |
| シナモンパウダー | 3g | 甘い香りと温かみ |
| ジンジャーパウダー | 2g | 後味の辛味と香り |
| クローブパウダー | 1g | 奥行き。強く出やすいため少量 |
| ブラックペッパー | 1g | 香りの輪郭。刺激を抑えた量 |
| **合計** | **15g** |  |

カルダモンを8g配合し、全体の半量以上を占める構成にする。すべて手持ち候補の粉末スパイスで作れるため、ホールスパイスの粉砕は必須にしない。

### 手順

1. 乾いた小皿またはボウルに、各スパイスを重量で量る。
2. 小さな泡立て器またはスプーンで、色と香りが均一になるまで混ぜる。
3. 少量を手のひらで温め、カルダモンの香りが主役になっているか確認する。
4. 乾燥した密閉容器へ移し、作成日と配合名を記録する。

## チャイへの使い方

- 1杯分のチャイに0.4gから加える。
- チャイ本体の個別スパイス（カルダモン、シナモン、クローブ、ジンジャー）は加えず、このマサラで置き換える。
- 水とマサラを先に煮出し、その後にアッサム茶葉と牛乳を加える。
- 香りが弱い場合だけ、0.1gずつ追加する。
- 砂糖は別に調整し、香りが弱いことを甘さで補わない。

### 使用量の目安

| 目的 | 1杯あたり | 使い方 |
| --- | ---: | --- |
| 軽く香りを足す | 0.4g | 毎日の基準。まずここから開始 |
| 標準的に使う | 0.5g | 甘い香りをしっかり感じる量 |
| 香りを強くする | 0.6gまで | 飲み疲れしないか確認して追加 |

## 調整方法

- カルダモンの香りが弱い場合は、次回配合でカルダモンを1g増やし、シナモンを1g減らす。
- 甘い香りが強すぎる場合は、カルダモンまたはシナモンを減らす前に、1杯あたりの使用量を0.1g下げる。
- クローブの香りが薬品のように感じる場合は、次回配合で半分にする。
- 刺激が強い場合は、ブラックペッパーとジンジャーを合計1g減らし、カルダモンを同量増やす。
- 毎日飲んで飽きる場合は、スパイスの種類を増やす前に使用量を0.4gへ戻す。

## 保存

- 完全に乾いた密閉容器へ入れる。
- 湿気、直射日光、高温を避け、冷暗所で保管する。
- 濡れたスプーンを容器へ入れない。
- 香りを重視する PoC のため、作成後1か月程度を評価期間の目安とする。
- 湿気て固まった、異臭がある、カルダモンの香りが明らかに弱くなった場合は使用せず、次の配合を作る。

## 設計意図

### カルダモンを主体にする理由

カルダモンは牛乳と紅茶に合わせたとき、砂糖を増やさなくても甘い印象を作りやすい。全体の半量以上にして、チャイ専用マサラとしての特徴を明確にする。

### 毎日飲んでも飽きにくくする理由

シナモン、クローブ、ブラックペッパーを強くすると香りが重くなりやすい。補助スパイスを少量に抑え、使用量を調整しやすくすることで、日による好みの変化にも対応する。

### 粉末スパイスを使う理由

手持ちの粉末スパイスをそのまま使い、毎日の調理にホールスパイスの計量や粉砕を追加しない。香りが不足する場合は、次の試作でホールカルダモンの導入を提案する。

## 初回評価項目

- カルダモンの香りが明確に主役になっているか
- 砂糖小さじ1の甘さ控えめチャイでも満足感があるか
- 0.4〜0.6gの使用量で調整しやすいか
- クローブ、ジンジャー、ブラックペッパーが強すぎないか
- 毎日数日続けても飽きにくいか
- 粉末が茶こしで扱いやすいか
- 1か月保管後も香りを維持できるか
- 次回に変更する要素を1つに絞れるか

初回試作の所感と改善内容は、調理・試飲後に `history/` 配下へスナップショットとして記録する。
