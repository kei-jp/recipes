# 毎週作れる定番ダルカレー（標準）

## 位置づけ

レンズ豆を使って、毎週無理なく作れることを優先した作り置き用ダルカレーである。
約6食分を一つの鍋で作り、冷蔵または冷凍して平日の主菜に使う。

レンズ豆は浸水なしで調理できるため標準とし、乾燥ひよこ豆へ置き換える場合は下処理を別に行う。
スパイスは [手持ちスパイス](../../ingredients/spices.md) を優先し、リスト外の材料は必要性と入手性を添えて提案扱いにする。

## 標準レシピ

### できあがり

- 約6食分
- 調理時間: 45〜60分程度
- 用途: 平日の昼食・夕食、作り置きの主菜
- 保存: 冷蔵または小分け冷凍

### 材料

#### 豆・野菜・ベース

| 材料 | 分量 | 役割・扱い |
| --- | ---: | --- |
| レンズ豆（乾燥） | 250g | 主な豆。洗って浸水せず使う |
| 玉ねぎ | 2個 | 甘みとうま味のベース |
| 人参 | 2本 | 野菜量と甘み |
| えのき | 1株 | うま味と食物繊維。石づきを落としてほぐす |
| なす | 2本 | とろみと食べ応え。乱切りにする |
| ブロッコリー | 1/2〜1株 | 仕上げに加え、食感と野菜量を補う |
| ホールトマト缶 | 1缶 | 酸味とうま味のベース |
| にんにく | 2片 | 香り付け。にんにく粉でもよい |
| 生姜 | 1片、またはジンジャー適量 | 香りと辛味。手持ちに合わせる |
| 水 | 1.2〜1.5L | レンズ豆の吸水量を見て調整する |
| 油 | 大さじ1 | 玉ねぎとスパイスを炒める |
| 塩 | 小さじ2から調整 | 最後に味見して加える |

#### スパイス

| スパイス | 分量 | 使用タイミング |
| --- | ---: | --- |
| クミンシード | 小さじ1 | 油で香りを出す |
| クミンパウダー | 小さじ1 | 野菜を炒めた後 |
| コリアンダーパウダー | 小さじ2 | 野菜を炒めた後 |
| ターメリック | 小さじ1/2 | 野菜を炒めた後 |
| ブラックペッパー | 小さじ1/2 | 煮込み前または仕上げ |
| カイエンペッパー | 少量〜小さじ1/2 | 辛味を付ける場合のみ |

### 下準備

1. レンズ豆を水で洗い、欠けた豆や異物を取り除く。
2. 玉ねぎ、人参、なす、にんにく、生姜を食べやすい大きさに切る。えのきは石づきを落としてほぐし、ブロッコリーは小房に分ける。
3. ホールトマトは缶の中で軽くつぶしておく。

### 手順

1. 大きめの鍋に油とクミンシードを入れ、中火で30秒ほど加熱する。焦がさず香りを油へ移す。
2. 玉ねぎ、にんにく、生姜を加え、玉ねぎが透き通るまで5分ほど炒める。
3. 人参となすを加え、なすに油が回るまで2〜3分炒める。
4. クミンパウダー、コリアンダー、ターメリック、ブラックペッパーを加え、30秒ほど炒める。焦げそうなら水を少量加える。
5. ホールトマト、レンズ豆、えのき、水1.2Lを加えて混ぜる。沸騰したら弱火にする。
6. 蓋を少しずらして20〜30分煮る。レンズ豆が柔らかくなるまで、ときどき鍋底を確認する。
7. 濃くなりすぎた場合は水を少量ずつ足す。豆が硬いまま水分がなくなった場合は、湯を足して追加加熱する。
8. レンズ豆が柔らかくなったらブロッコリーを加え、食感を残すため5分ほど加熱する。
9. 塩を少量ずつ加えて味を整える。辛味を付ける場合はカイエンペッパーを最後に加える。
10. 火を止め、冷めると濃くなることを考慮して、少しゆるめの濃度で仕上げる。
11. 浅い保存容器へ移して粗熱を早く取り、1食分ずつ保存する。

> 最初から水を全量入れて煮詰め切ろうとせず、豆の柔らかさと濃度を見ながら水を足すと失敗しにくい。

## 乾燥豆へ置き換える場合

### ひよこ豆を使う場合

乾燥ひよこ豆はレンズ豆と同じ手順で直接煮込まない。前日に浸水し、別鍋で柔らかくなるまで下茹でしてから使う。

1. 乾燥ひよこ豆をたっぷりの水に8時間以上浸ける。
2. 浸水水を捨て、新しい水で指でつぶせる程度まで下茹でする。
3. 標準レシピのレンズ豆250gの代わりに、茹でたひよこ豆500〜600gを使う。
4. 手順5で加える水は700〜900mlから始め、野菜と豆がなじむ濃度へ調整する。
5. ひよこ豆を加えた後、弱火で15〜20分煮て味をなじませる。

ひよこ豆版は粒感が残りやすく、レンズ豆版より水分が少ないカレーになる。作り置きで固く感じる場合は、食べる直前に水または湯を加える。

### その他の乾燥豆

豆の種類によって浸水時間、下茹で時間、吸水量が異なるため、必ず別茹でして柔らかさを確認してから加える。
レンズ豆用の水量と煮込み時間をそのまま流用しない。

## 置き換えと購入提案

### 野菜

えのき、なす、ブロッコリーを標準の野菜構成とする。なすは煮込むことでカレーになじみ、えのきはうま味と食物繊維を補い、ブロッコリーは後半に加えて食感を残す。
水分の多い野菜を増やす場合は、最初の水を少なめにする。

オクラは今回は標準材料に含めず、夏季やとろみを増やしたい場合の任意候補とする。使う場合は6〜8本を輪切りにし、ブロッコリーと同じタイミングで加える。粘りが出るため、まずは少量で評価する。

### スパイス

クミンを増やすと香ばしさ、コリアンダーを増やすと丸み、ブラックペッパーを増やすと後味の辛味が出る。
カイエンペッパーは最初から多く入れず、仕上げで調整する。

カルダモン、シナモン、ガラムマサラなどを加える案もあるが、毎週用の基本配合では必須にしない。追加する場合は、香りの変化と購入・補充コストを確認して提案扱いにする。

### たんぱく質の追加

豆だけでは一食のたんぱく質量が不足すると感じる場合、ゆで卵、ヨーグルト、鶏肉などを添える案を検討する。
本体の材料を増やしすぎず、食べ方で調整できるようにする。

## 保存・再加熱

- 粗熱が取れたら冷蔵庫へ入れ、常温に長時間置かない。
- 冷蔵保存は3日以内を目安にし、取り分けには清潔な器具を使う。
- 冷凍する場合は1食分ずつ小分けし、容器に日付を記録する。
- 食べる分だけ鍋または電子レンジで中心まで十分に再加熱する。
- 再加熱後に濃すぎる場合は、水または湯を少量加えて伸ばす。

## 設計意図

### レンズ豆を標準にする理由

レンズ豆は浸水や長い下茹でが不要で、毎週の調理負担を抑えやすい。煮崩れて自然なとろみが出るため、少ない油でも満足感を作りやすい。

### 健康とコストの両立

豆と野菜を主材料にし、油を大さじ1に抑える。ホールトマト缶、玉ねぎ、人参、えのき、なす、ブロッコリーなど入手しやすい材料を中心にし、特殊な調味料を必須にしない。

### 初心者が失敗しにくい設計

スパイスを短時間だけ炒め、豆と水を加えて煮る一鍋構成にする。ブロッコリーだけは仕上げに加えて食感を残す。水分は後から足せるため、最初から完璧な量を計量しなくても調整できる。

## 初回評価項目

- 6食分として量と濃度が適切か
- レンズ豆が指定時間で柔らかくなるか
- 野菜の量と食べ応えが十分か
- 辛さとスパイス感が毎週食べても重すぎないか
- 冷蔵3日目・冷凍後の食感が許容範囲か
- 材料費と調理時間が継続可能な範囲か
- 次回に変更する要素を1つに絞れるか

初回調理の所感と改善内容は、調理後に `history/` 配下へスナップショットとして記録する。
