# 平日作り置き用スパイスカレー（標準）

## 位置づけ

この文書は、平日の昼食向けに継続して作るスパイスカレーの標準レシピである。
完成形を固定するのではなく、調理結果と改善理由を蓄積し、次回の実験の基準として使う。

本場の味の再現よりも、次の性質を優先する。

- 一度に約5食分を作り置きできる
- 野菜とたんぱく質を一皿で摂取できる
- 冷凍後も食べやすい
- 基本スパイスの配合を自分で調整できる
- 特定の商品やカレールーに依存しない

## 標準レシピ

### できあがり

- 約5食分
- 用途: 平日の昼食用
- 保存: 小分けして冷凍することを前提とする

### 手順

1. 玉ねぎをブレンダーにかける。人参、ナス、ブロッコリー、えのき、鶏もも肉は食べやすい大きさに切る、野菜類は好みでブレンダーにかける。
2. 鍋に油を入れて熱し、クミンシード小さじ1を加えてテンパリングする。焦がさず、香りを油へ移す。
3. 玉ねぎを加え、焦げ付かないように混ぜながら飴色になるまで炒める。
4. ホールトマト、人参、ナス、えのき、鶏ミンチ、鶏もも肉を加える。
5. クミンパウダー、コリアンダーパウダー、ターメリック、カイエンペッパーを加え、全体をよく混ぜる。煮込みに必要なら水を加える。
6. 弱火で20〜30分煮込み、鶏肉に十分火を通す。鍋底が焦げ付かないよう、ときどき混ぜる。
7. ブロッコリーを加え、食感を残すため5分程度加熱する。
8. 火を弱めて味噌を溶き入れる。最後に塩で味を調整する。
9. 粗熱を取り、1食分ずつ小分けして冷凍する。

> 鶏肉は中心部まで十分に加熱する。保存容器へ移した後は室温に長時間放置せず、速やかに冷まして冷凍する。

## 設計意図

### 作り置きと野菜摂取の両立

玉ねぎとトマトをベースに、根菜、ナス、きのこ、ブロッコリーを組み合わせる。
野菜の種類を固定しすぎず、在庫や季節に合わせて入れ替えられる構成とする。
一方、冷凍と再加熱に適するかは、置き換え時の評価項目に含める。

### 2種類の鶏肉を使う理由

鶏ミンチはカレー全体へうま味とたんぱく質を分散させ、鶏もも肉は一食ごとの食べ応えを補う。
次回以降に肉を変更する場合は、味だけでなく脂質、満足感、冷凍後の食感も比較する。

### スパイスを分離して管理する理由

市販のカレールーや配合済みの赤缶に依存せず、クミン、コリアンダー、ターメリック、カイエンを個別に量る。
これにより、辛さ、香り、色を独立して調整し、変更の効果を追跡できる。

### ブロッコリーと味噌を最後に加える理由

ブロッコリーは長時間煮込まず、仕上げに加えて食感と色を残す。味噌も最後に加え、隠し味としての風味を確認しながら調整する。

## 初回の評価

| 評価項目 | 結果 |
| --- | --- |
| 総合 | 初回として十分満足できる味 |
| 辛さ | カイエンペッパー小さじ2では辛すぎた |
| 香り | 配合済みスパイス（赤缶）を追加する必要は感じなかった |
| 作り置き適性 | 十分に向いていそう |
| 収量 | 約5食分 |
| 再調理意向 | 改善しながら標準として継続する |

冷凍後の味、食感、水分分離については、実食後に評価を追記する。

## 次回の改善候補

一度に変更する要素は原則1つとし、比較できる状態を保つ。

| 優先度 | 候補 | 理由・検証方法 |
| ---: | --- | --- |
| 1 | カイエンペッパーを減らす | 初回の明確な問題点。まずカイエンだけを減らし、辛さと香りのバランスを比較する |
| 2 | ターメリックを少し減らす | 苦みや土っぽさへの影響を確認する。カイエンの適量を決めた後、単独で変更する |
| 3 | 油の使用量を計量する | 標準レシピの再現性と、栄養・保存状態の比較可能性を高める |
| 4 | 牛豚ミンチと比較する | うま味、脂質、価格、冷凍後の食感、満足感を鶏ミンチ版と比較する |
| 5 | ガラムマサラの導入を検討する | 現状で赤缶は不要だったため必須ではない。仕上げの香りに不足を感じた場合のみ少量で評価する |
| 6 | ひよこ豆を加え、チャナマサラ系へ派生する | 植物性たんぱく質、食物繊維、コスト、腹持ちを評価する |
| 7 | ヨーグルトを使い、軽めのコルマ系へ派生する | 辛さを抑えた別系統として、酸味、まろやかさ、冷凍時の分離を評価する |

## 今後の発展

標準版では、まず辛さと基本分量の再現性を安定させる。派生版は標準版の評価軸を引き継ぎ、別文書として管理する。

- **たんぱく源の展開**: 牛豚ミンチ、ひよこ豆、レンズ豆
- **野菜の展開**: 季節と在庫に応じた置き換え
- **香りの展開**: ガラムマサラなどを小容量から評価
- **味の系統の展開**: チャナマサラ系、軽めのコルマ系
- **運用の改善**: 1食分の重量、冷凍期間、再加熱方法、冷凍後の評価を記録

派生を増やす前に標準版を十分に改善し、変更点と理由を改善履歴へ残す。
